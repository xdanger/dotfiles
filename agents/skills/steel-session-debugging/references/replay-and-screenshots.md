# Replay And Screenshots

Replay artifacts help verify the timeline visually.

## Useful Surfaces

- session viewer URL for live or recent state
- player URL for past replay
- HLS recording when available
- screenshots taken by the agent or browser command
- final page URL and visible text from the last snapshot

## What To Look For

- a modal or cookie banner covering the intended target
- a challenge page after a successful-looking click
- wrong tab or popup focus
- form validation messages
- missing uploaded file or disabled submit button
- UI state that diverges from the trace's intended action

## Sharing Guidance

Do not export screenshots or recordings outside the local machine unless the user asks. Screenshots may contain personal data, login state, or page content under contract.
